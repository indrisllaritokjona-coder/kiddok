### Fix Applied During Sprint

**Test:** `should show error on invalid PIN`  
**Problem:** Test looked for text `text=Invalid, gabim, pasaktë` but the actual error message rendered by the component is `p.text-red-600` with i18n key `pin.invalidCredentials` — which produces `Kodi ose fjalëkalimi është i pasaktë.` (SQ) or `Invalid username or password.` (EN). Neither of those literal strings matches the test's locator.

**Fix:** Changed the locator to `page.locator('p.text-red-600').first()` — targeting the red error paragraph that is actually rendered by the component's `@if (errorMsg())` block.

```diff
- const errorMsg = page.locator('text=Invalid, gabim, pasaktë').first();
+ const errorMsg = page.locator('p.text-red-600').first();
```
